# PikaDB 文档

这是 [Pika 项目](https://github.com/OpenAtomFoundation/pika) 的使用者所可以查阅的官方文档，里面涵盖了诸多要点、经验与框架。

## 如何构建文档网站？

```shell
git clone https://github.com/pikiwidb/website.git
cd website
# 请确保你安装了 Vite 软件
npm run docs:dev
```

# PikiwiDB(PikiDB) Document

The document of [PikiwiDB](https://github.com/OpenAtomFoundation/pika).

## How to build?

```shell
git clone https://github.com/pikiwidb/website.git
cd website
# Make sure you have installed Vite
npm run docs:dev
```
